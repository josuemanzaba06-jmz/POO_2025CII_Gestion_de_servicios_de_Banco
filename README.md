DESCRIPCIÓN DEL PROGRAMA

Sistema de gestión bancaria creado en Python por medio de Programación Orientada a Objetos (POO). 
Para administrar diversas clases de servicios bancarios, como los depósitos, las transferencias, los pagos de servicios y los retiros, 
este proyecto aplica los principios básicos de la POO: herencia, encapsulamiento y polimorfismo.
